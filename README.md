# Susan Collins Vote Tracker

An interactive newsroom tool that ranks every one of Sen. Susan Collins' votes by
importance — letting readers decide what "important" means via four weighted factors
(closeness, decisiveness, party breaks, topic salience).

- `collins-votes-tool.html` — the self-contained reader tool (open in any browser)
- `scrape_collins_votes.py` — pulls her full voting record from senate.gov
- `HANDOFF.md` — plain-English guide to running and publishing it

Data source: U.S. Senate official roll-call record (senate.gov).
